DATA FLOW DIAGRAM
=================

User Input (Text or URL)
      ↓
    validate_article_text()
      ↓
   get_llm_handler(provider)
      ↓
   ┌─────────────────────────────────┐
   │  Chain-of-Thought Analysis      │
   │                                 │
   │  1. detect_biases()             │
   │     → Returns: biases[], score  │
   │                                 │
   │  2. neutralize_article()        │
   │     → Returns: neutralized text │
   │                                 │
   │  3. generate_perspective()      │
   │     → Returns: left/right text  │
   └─────────────────────────────────┘
      ↓
   save_analysis() → Supabase
   save_detected_biases() → Supabase
   save_perspective() → Supabase
      ↓
   display_analysis_results()
      │
      ├── Tab: Original
      ├── Tab: Neutralized
      ├── Tab: Bias Analysis
      └── Tab: Perspectives

QUICK REFERENCE
===============

To Run:
  pip install -r requirements.txt
  streamlit run app.py

To Get Started:
  1. Open browser → http://localhost:8501
  2. Paste API key in sidebar
  3. Click "Test Selected Provider"
  4. Paste article or URL
  5. Click "Analyze Article"
  6. View results in tabs

Supported Providers:
  • Claude (Anthropic): claude-3-5-sonnet
  • OpenAI: gpt-4-turbo
  • Gemini (Google): gemini-pro

Database:
  • Supabase (PostgreSQL)
  • Pre-configured in .env
  • RLS policies enabled
  • Session-based access control

Documentation:
  • QUICKSTART.md - 5 minutes to start
  • README.md - Complete guide
  • TESTING_GUIDE.md - How to test
